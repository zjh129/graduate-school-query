# 院校数据维护说明（写给非技术人员）

> 本应用不依赖数据库，所有数据来自 `public/data/` 下的**纯文本文件**。  
> 修改完保存，刷新浏览器就生效，不需要重新打包。

## 📁 文件位置

```
public/data/
├── schools.csv     ← 主数据源（推荐 Excel / WPS / Numbers 编辑）
├── schools.md      ← 备选（VSCode / Typora / 记事本都能编辑）
└── schools.json    ← 程序用，不建议手改
```

应用启动时按以下顺序自动找，**找到第一个就用**，不用三个都准备：

1. `schools.csv`
2. `schools.md`
3. `schools.json`
4. `schools.xlsx`
5. `schools.xls`

## 📋 字段含义

| 字段 | 必填 | 说明 | 取值示例 |
|---|---|---|---|
| **学校标识码** | ✅ | 教育部 5 位国标代码 | `4111010001` |
| **学校名称** | ✅ | 完整校名 | `北京大学` |
| **投档分数** | ⬜ | 留空 = 仅展示，按分数匹配时不参与 | `621` 或留空 |
| **最低位次** | ⬜ | 投档最低位次，留空显示 `-` | `1234` |
| **省份** | ✅ | 院校所在省份 | `北京市` |
| **城市** | ✅ | 院校所在城市（不含省份前缀） | `北京市` / `昌吉回族自治州` |
| **院校层次** | ✅ | `本科` 或 `专科(高职)` | `本科` |
| **院校性质** | ✅ | `公办` / `民办` / `中外合作办学` / `内地与港澳台地区合作办学` | `公办` |
| **院校特色** | ⬜ | 多选用 `|` 分隔 | `985\|211\|双一流` |
| **招生类型** | ✅ | `普通类` / `中外合作办学` / `校企合作` / `艺术类` / `体育类` | `普通类` |
| **选科要求** | ✅ | `不限` / `物理` / `历史` / `物理+化学` / `物理+化学+生物` / `物理+地理` / `历史+地理` / `历史+政治` | `不限` |
| **主管部门** | ⬜ | 教育部 / 各省教育厅等 | `教育部` |

## ✏️ 编辑方法

### 方法 A：用 Excel / WPS / Numbers 编辑 CSV（最简单）

1. 双击打开 `public/data/schools.csv`
2. 像普通 Excel 一样编辑
3. **另存为**：文件类型选 `CSV UTF-8 (逗号分隔) (*.csv)`
   - ⚠️ **千万不要**直接保存为 `.xlsx`，会丢格式！
   - ⚠️ Mac Numbers 需「导出为 → CSV」

### 方法 B：用 VSCode / Typora / 记事本编辑 Markdown

`schools.md` 就是个表格，可读性最好：

```markdown
| 学校标识码 | 学校名称 | 投档分数 | 最低位次 | 省份 | 城市 | 院校层次 | 院校性质 | 院校特色 | 招生类型 | 选科要求 | 主管部门 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 4111010001 | 北京大学 |  |  | 北京市 | 北京市 | 本科 | 公办 | 985\|211\|双一流 | 普通类 | 不限 | 教育部 |
```

### 方法 C：从原始名单批量生成

如果拿到一份「全国普通高等学校名单 .xls」：

```bash
node scripts/gen-data-sources.mjs
```

会从 `data/` 下最大的 xls 自动生成三个数据源。  
然后只需要手动补 **投档分数 + 最低位次 + 院校特色** 这三列即可（其他已自动填好）。

## ❓ 常见问题

**Q: 改完没生效？**  
A: 检查 `public/data/` 下文件命名是不是 `schools.csv`（小写、复数 s）。硬刷新（Ctrl+Shift+R）。

**Q: CSV 打开后中文乱码？**  
A: 用 VSCode 右下角切到 UTF-8。Excel 打开正常但再保存会丢 BOM，建议用 WPS 或 Numbers。

**Q: 「院校特色」单元格怎么填多个？**  
A: 用 `|` 分隔，例如 `985|211|双一流`。CSV 里单元格内有逗号才需要引号，`|` 不需要。

**Q: 投档分数这一列留空会怎样？**  
A: 留空 = 该校**不出现在匹配结果**里（名单型数据），等分数出来后再补。

**Q: 同一所院校有多个选科要求怎么办？**  
A: 复制一行，改选科要求列。新高考可拆成 `物理`、`物理+化学` 等多行。

**Q: 哪里能拿到最新全国院校名单？**  
A: 教育部官网 → 「全国高等学校名单」，下载 xls 放 `data/` 目录重跑生成脚本。
